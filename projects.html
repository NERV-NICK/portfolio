Lime Click

Lime Click - это увлекательная игра-кликер, разработанная для Telegram. Она создана с использованием Python Flask, JavaScript, HTML, CSS и SQL. Это интерактивное приложение, которое предлагает пользователям кликать и набирать очки, в то время как они наслаждаются динамичным и захватывающим геймплеем.

Password generator

Генератор паролей - это удобное приложение с графическим интерфейсом, разработанное для операционной системы Windows с использованием PyQt6. Это инструмент, который обеспечивает безопасность ваших данных, создавая надежные пароли.

Random numbers generator

Мобильное приложение, разработанное на Kivy. Это инструмент, который случайным образом генерирует числа в указанном диапазоне.

Amusement Park

Веб-сайт для парка развлечений, разработанный на FastAPI, с функцией онлайн-бронирования билетов. Это удобное решение для посетителей, позволяющее им заблаговременно планировать свой день и избегать очередей.
